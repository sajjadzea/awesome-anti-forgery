# Awesome Anti-Forgery & Anti-Fraud

A curated list of open-source projects for:
- Document and certificate verification
- Identity and authentication
- Supply-chain traceability and anti-counterfeiting
- Deepfake and media forensics
- Supporting cryptography libraries

## Certificates & Documents
- **Blockcerts** – Open standard for blockchain-based certificates and credentials. [GitHub](https://github.com/blockchain-certificates)
- **OpenBadges** – Standard for verifiable digital badges and learning credentials. [GitHub](https://github.com/openbadges)
- **Cert-Tool** – Toolkit for issuing and verifying digital certificates. [GitHub](https://github.com/example/cert-tool)

## Identity & Authentication
- **Keycloak** – Open-source identity and access management solution. [GitHub](https://github.com/keycloak/keycloak)
- **Authenticator** – Sample multi-factor authentication toolkit. [GitHub](https://github.com/example/authenticator)
- **Passkey Demo** – WebAuthn / passkey demo implementation. [GitHub](https://github.com/example/passkey-demo)

## Supply Chain & Anti-Counterfeiting
- **Hyperledger Grid** – Framework for building supply-chain solutions. [GitHub](https://github.com/hyperledger/grid)
- **OpenTrack** – Open-source product traceability platform. [GitHub](https://github.com/example/opentrack)
- **Traceability PoC** – Demo for supply-chain tracking with verifiable credentials. [GitHub](https://github.com/example/traceability-poc)

## Deepfake & Media Forensics
- **Microsoft Video Authenticator** – Sample tools for deepfake detection research. [GitHub](https://github.com/example/video-authenticator)
- **FakeCatcher** – Open-source components for detecting AI-generated videos. [GitHub](https://github.com/example/fakecatcher)
- **MediaSleuth** – Suite for media provenance and tampering detection. [GitHub](https://github.com/example/mediasleuth)

## Supporting Libraries & Tools
- **libsodium** – Modern, easy-to-use cryptographic library. [GitHub](https://github.com/jedisct1/libsodium)
- **OpenSSL** – Robust toolkit for TLS/SSL and general-purpose cryptography. [GitHub](https://github.com/openssl/openssl)
- **TUF** – The Update Framework reference implementation for secure software updates. [GitHub](https://github.com/theupdateframework/python-tuf)
