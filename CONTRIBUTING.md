# Contributing

Thanks for your interest in contributing! Follow these steps to add a new project:

1. **Fork** this repository and create a new branch for your changes.
2. Add your project under the appropriate section in `README.md` using the format:
   - `**Project Name** â€“ short description. [GitHub](https://github.com/your-org/your-project)`
3. Keep descriptions concise and ensure the GitHub link points to an open-source project.
4. If relevant, add an entry to `projects.json` with `name`, `category`, `description`, and `url`.
5. Open a **pull request** describing your additions and any context that will help reviewers.

Thank you for helping curate this list!
