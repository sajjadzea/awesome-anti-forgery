# نقشه راه مخزن

این نقشه راه، اولویت‌های توسعه برای ارتقای فهرست به سطح «۱٪ برتر» را مشخص می‌کند.

## 1. پوشش حوزه‌های نوظهور
- افزودن پروژه‌های ضدجعل در NFT و دارایی‌های دیجیتال.
- اضافه کردن ابزارهای watermarking برای مدل‌های تولیدی و محتوای مولد.
- گسترش پروژه‌های فورنزیک صوت و گفتار برای تشخیص جعل صوتی.

## 2. تنوع پشته و زبان
- پوشش کتابخانه‌های رمزنگاری در زبان‌های Go، Rust و JavaScript در کنار C/Python.
- اضافه کردن نمونه‌های SDK برای موبایل (Android/iOS) جهت امضای دیجیتال و هویت غیرمتمرکز.

## 3. کیفیت داده و مثال‌های ادغام
- ارائه مثال‌های ادغام (integration examples) در پوشه `examples/` برای اتصال سریع به Keycloak، OpenSSL، Tink و پروژه‌های Fabric.
- افزودن بنچمارک‌های ساده برای مدل‌های تشخیص دیپ‌فیک با استفاده از FaceForensics++.

## 4. حاکمیت و انطباق
- مستندسازی سیاست گزارش آسیب‌پذیری در `SECURITY.md` و به‌روزرسانی دوره‌ای.
- افزودن الگوی انتشار (release template) برای نسخه‌های دوره‌ای فهرست.

## 5. خودکارسازی و آزمون
- بهبود CI با لایه کش برای درخواست‌های HTTP در `--check-urls` و گزارش خطاهای خوانا.
- افزودن lint خودکار برای سبک Awesome List و لینک‌های معیوب.
