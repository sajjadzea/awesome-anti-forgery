- Certificates & Documents

## Highlights
- Tables of vetted GitHub projects with concise Persian descriptions in [`README.md`](README.md).
- Machine-readable catalog in [`projects.json`](projects.json) validated by CI for required fields and working links.
- Contributor guides, schema reference, and ready-to-use issue/PR templates to keep the list consistent.

## Contributing
1. Add the project to the relevant table in `README.md` (Persian) and to `projects.json` with full metadata.
2. Ensure categories, tags, and licenses follow [`docs/SCHEMA.md`](docs/SCHEMA.md).
3. Run `python scripts/validate_projects.py --check-urls` before opening a pull request.
